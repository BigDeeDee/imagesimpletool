## ✨ 特性

- 🖼️ **批量上传**：一次选择多张图片（支持 JPG/PNG/WebP/BMP/GIF）
- ✂️ **统一裁剪**：任意图片设为参考框 → 拖拽/缩放/旋转 → 一键应用到所有图片
- 🎨 **格式转换**：输出可选 JPEG / PNG / WebP，JPEG/WebP 支持画质调节
- 📦 **打包下载**：所有处理后图片可打包为 ZIP 文件，同时支持单张点击下载
- 🌐 **中英文双语**：界面一键切换中英文，适合国内外用户
- 📱 **响应式设计**：适配桌面与平板，简洁现代的玻璃态 UI
